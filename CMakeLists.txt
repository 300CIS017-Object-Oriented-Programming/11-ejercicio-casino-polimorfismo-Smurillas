# Usar C++17

# Ejecutable y archivos fuente

        Model/Casino.cpp
        Model/DosColores.cpp
        Model/Mayor13.cpp
        Model/Slots.cpp
        Model/Jugador.cpp
        Model/ParImpar.cpp
)
